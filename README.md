# Real-Time Stock Screener Dashboard

## Overview
This project is a real-time stock screener dashboard built using Next.js 14, React 18, TypeScript, and Tailwind CSS.

## Objective
The goal of this project is to build a high-performance stock screening dashboard that can handle 5000+ stock records with fast filtering, sorting, live price updates, and chart visualization.

## Tech Stack
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- TanStack Table
- TanStack Virtual
- Zustand
- Lightweight Charts

## Current Features
- Next.js project setup
- GitHub repository setup
- Basic dashboard layout
- Mock stock data generator
- 5000+ stock record support
- Basic stock table preview

## Performance 
Performance validation confirms smooth handling of 5000+ stock records with virtualized rendering and sub-200ms filtering.
